# Longest Palindromic Substring

## Problem statement:
Given a string s, return the longest palindromic substring in s.

## Examples:
```py
Input: s = "babad"
Output: "bab"
Note: "aba" is also a valid answer.
```
```py
Input: s = "cbbd"
Output: "bb"
```
```py
Input: s = "a"
Output: "a"
```
```py
Input: s = "ac"
Output: "a"
```

## Constraints:
- 1 <= s.length <= 1000
- s consist of only digits and English letters (lower-case and/or upper-case),
